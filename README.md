A GenAI project that analyzes research papers using Retrieval-Augmented Generation (RAG).

## Features
- Ask questions about a research paper
- Generate summaries
- Extract key contributions
- Explain content in simple terms
- Streamlit web interface

## Tech Stack
- Python
- LangChain
- Hugging Face Transformers
- FAISS (Vector DB)
- Streamlit

## Note
This project uses local models, so performance depends on system resources.

## Future Improvements
- Faster models
- Better answer accuracy
- Support multiple papers
